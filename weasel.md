# 安装方法〔Windows〕

在 Windows 上安装南京话拼音输入法，有两大步骤：（1）安装小狼毫输入法；（2）添加南京话拼音的方案和码表。

## 安装小狼毫输入法

从 Rime 的官网下载“小狼毫输入法”。下面的截图来自版本 0.9.30，部分界面与最新版本略有差异。

> https://rime.im/download

![image-a1]

运行安装程序。输入法程序需要比较高的系统权限，如果被某些杀毒软件拦截，请允许放行。
